🔌 Type of Connections
Config File
Manually edit claude_desktop_config.json. Add server definitions with command, args, and env variables. Best for custom or third-party servers.
Connectors
Built-in UI in Claude Desktop. Click to add, OAuth login handled automatically. Best for supported apps — no JSON editing required.
🔗 What Are Connectors?
A Connector is a built-in feature that links Claude to MCP servers automatically, without manual setup or configuration.
Designed for non-technical users who want Claude to talk to their apps
No running servers, no editing JSON, no transport config
Handles OAuth authentication (Google, GitHub, etc.) automatically
Think of it as the "App Store" for MCP servers — click-based, pre-curated
easysafeconsistentOAuthno-config
⚠️ Why Not Use Connectors Always?

Curated & Managed

Connectors are officially built, hosted, and maintained by Anthropic. They have OAuth, security, rate-limits, and stability — but Anthropic can't review every possible server. Doesn't scale.

MCP is an Open Standard

MCP is designed so anyone can write a server. Forcing everything through Connectors would close the ecosystem and make you dependent on Anthropic to approve/publish servers.

🎬 Demo Highlights
manim-server

Local connector for creating math animations. Prompt Claude to generate a linear algebra matrix transformation animation — Claude calls the MCP tool automatically.

Google Drive

Web connector. Click Connect, sign in via OAuth, and Drive search is instantly available in the tools menu. Prompt: "From Drive" context suggestion appears.

Twitter MCP

Via config file using npx @enescinar/twitter-mcp. Requires API Key, API Secret, Access Token, Access Token Secret from X Developer portal.

Weather MCP

GitHub: adhikasp/mcp-weather. Uses uvx command + AccuWeather API key. Provides hourly/daily forecasts.

📄 Config File — claude_desktop_config.json

Full example combining manim-server, Twitter MCP, and Weather MCP:

{
  "mcpServers": {
    // Local: manim animation server
    "manim-server": {
      "env": {
        "MANIM_EXECUTABLE": "/Library/Frameworks/Python.../bin/manim"
      }
    },
    // Remote: Twitter via npx
    "twitter-mcp": {
      "command": "npx",
      "args": ["-y", "@enescinar/twitter-mcp"],
      "env": {
        "API_KEY": "<your-api-key>",
        "API_SECRET_KEY": "<your-secret>",
        "ACCESS_TOKEN": "<your-token>",
        "ACCESS_TOKEN_SECRET": "<your-token-secret>"
      }
    },
    // Remote: Weather via uvx
    "weather": {
      "command": "uvx",
      "args": ["--from", "git+https://github.com/adhikasp/mcp-weather.git", "mcp-weather"],
      "env": {
        "ACCUWEATHER_API_KEY": "<your-accuweather-key>"
      }
    }
  }
}
