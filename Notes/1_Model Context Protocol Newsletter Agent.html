06 · Step 6 — Designing Phase (Agent 3)

Email Template Builder Agent Flow

Claude→ Final Draft (.md) Design Specs → HTML email + .txt fallback → Desktop/output/

Layout Rules

  • Max width: 600px, centered
  • Tables only for layout (Outlook safe)
  • Mobile responsive via media queries
  • Section headers with thin dividers

Styling Rules

  • Inline CSS for all critical styles
  • BG: #FFFFFF · Text: #111111
  • Accent: #2F80ED
  • System font stack (safe fonts)
  • VML button for Outlook

QA Checklist — Self-verify Before Finalizing

  • Every section has at least one link
  • No empty sections (omit if no content)
  • No images wider than 600px
  • Buttons render on Outlook (VML)
  • Mobile stacking validated for Quick Updates & Products
  • Original links intact — no tracked/shortened URLs
  • Spellcheck all headings & button labels

Plain-Text Fallback

  • Headings → ALL CAPS with underlines
  • Links on their own line (full URL)
  • Bullets as -
  • Lines wrapped to ~78 characters
